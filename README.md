# Slider Project

Um projeto de **slider/carrossel interativo** desenvolvido com **HTML, CSS e JavaScript**, focado em animações suaves, navegação dinâmica e design responsivo.

## 🚀 Demonstração

O projeto apresenta destinos turísticos com imagens em destaque, miniaturas navegáveis e transições animadas, criando uma experiência visual moderna e dinâmica.

## 📸 Preview

![Slider Preview](./assets/slider-preview.png)

## ✨ Funcionalidades

* Slider de imagens em tela cheia
* Navegação com botões **Next** e **Back**
* Miniaturas sincronizadas com o slide principal
* Animações suaves para transições
* Efeitos visuais nos textos
* Layout moderno e responsivo

## 🛠️ Tecnologias utilizadas

* HTML5
* CSS3
* JavaScript (Vanilla)

## ▶️ Como executar

1. Clone o repositório:

```bash
git clone git@github.com:kaiqueRicci/sliderProject.git
```

2. Acesse a pasta do projeto:

```bash
cd sliderProject
```

3. Abra o arquivo `index.html` no navegador.

## 📖 Aprendizados

Durante o desenvolvimento deste projeto, foram praticados conceitos importantes como:

* Manipulação do DOM
* Eventos com JavaScript
* Uso de `appendChild()` e `prepend()`
* Criação de animações com `@keyframes`
* Organização de layout com Flexbox e Grid
* Controle de estados com classes CSS
* Estruturação de componentes visuais interativos
